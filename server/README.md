## STEPS TO RUN SERVER

1. Move to ~/react-cryptochat/server
2. run 
    ```bash 
    npm install
    ```
4. Create an .env file. 
5. Add 
    ```bash
   PORT=4000
   ```
1. Save the file and start server: 
    ```bash
    npm run dev
    ```

1. Server should now be running and console print: **"LISTENING ON PORT <port>"**
